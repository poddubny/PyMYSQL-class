# PyMYSQL-class
A Pymysql class for developers to connect python with mysql

### Min  Requirement :
  PyMySQL
  
PyMySQL can be installed by following command
```
$ pip install PyMySQL
```
## Running the tests
```
db_obj = db() //Cretae a object
inserting_arr = {"key1":"value1", "key2":"value2" ....}

```
